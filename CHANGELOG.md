# 更新日志

## v1.1.0 - 2026-06-22

### 拍摄地点

- GPS 坐标会尝试通过手机系统的反向地理编码转换为可读地点，例如省、市、区和街道。
- 记忆卡和查找结果现在单独显示“拍摄地点”与“室内线索”，不再把真实地点和“桌面、抽屉”等视觉线索混在一起。
- 查找命中带有地点的记录时，会优先使用“某物可能在某地点的某室内线索”这种组合表达。

### 查找体验

- 提交查找后会立即进入“小龟正在翻找记忆”的过渡页，等待结果期间不再像界面卡住。

## v1.0.0 - 2026-06-20

首个可用测试版本。

### 物品记忆

- 支持在 App 内拍照保存，不写入系统相册。
- 支持 MiMo 多模态识别主要物品、别名、画面场景、位置线索和搜索摘要。
- 支持记录拍摄时间、GPS 坐标和定位精度。
- 拍完照片后自动后台识别；退出或关闭 App 后，待识别记录仍会保留。
- 提供识别进度页，可查看识别详情并手动重试失败记录。
- 支持删除单条记忆，同时清理照片、缩略图和记录。

### 查找与设置

- 支持自然语言查找，并返回匹配照片、理由、拍摄时间和位置线索。
- 支持填写 MiMo API Key，并提供带图片的连通性测试。
- API Key 只保存于手机本机安全存储。

### 验证

- 已在 Android 模拟器完成“拍照、自动识别、查看记忆、自然语言查找”的完整链路验证。
- `flutter test` 通过，26 项测试全绿。
- Android arm64 release APK 构建通过。
