# 小龟寻物

小龟寻物是“小龟”系列的第二款 Android App，和小龟听书同属一个系列。

它用来拍下东西放在哪里，并在之后用自然语言把它找回来。

## 下载

当前版本：`v1.0.0`

前往 Release 下载 APK：

[下载小龟寻物 v1.0.0](https://github.com/earthking11/xiaogui-xunwu/releases/tag/v1.0.0)

安装包名称：

```text
xiaogui-xunwu-v1.0.0-arm64-debug.apk
```

说明：这是面向 arm64 Android 手机的测试版 APK，不是应用商店正式签名版。

完整版本变化见：[更新日志](./CHANGELOG.md)

## 主要功能

- 拍照后，照片只保存在 App 私有目录，不混入系统相册。
- 使用 MiMo `mimo-v2.5` 自动识别主要物品、别名、场景、室内位置线索和搜索摘要。
- 自动记录拍摄时间、GPS 坐标和定位精度。
- 支持查看识别进度，失败后可手动重试。
- 支持用自然语言查找物品，返回当时的照片、匹配理由和位置线索。
- 记忆库支持删除单条内容，同时清理原图和缩略图。
- 没有 API Key、网络异常、GPS 失败或 AI 失败时，照片不会丢失，记录可稍后重试。

## 如何使用

1. 打开 App 后允许相机权限。
2. 点击相机按钮拍下物品所在位置。
3. 首次使用时，进入右上角设置填写 MiMo API Key，并用图片连通性测试确认可用。
4. 稍等识别完成后，可在“识别进度”中查看物品、场景和位置线索。
5. 在首页输入例如“我的放大镜在哪里”，即可查找对应照片和记录。

## 关于 MiMo API Key

小龟寻物不会内置任何 API Key。你需要自己准备 MiMo API Key，并在 App 内填写。

Key 只保存在手机本机的安全存储中；项目源码、README 和 APK 中都不包含真实 Key。

连通性测试会发送一张应用内测试图片。识别一张物品记忆时，照片与可用的 GPS 坐标会发送给 MiMo，用于生成记忆卡。

## 隐私和权限

- 照片保存在 App 私有目录，不写入系统相册。
- 相机权限只用于拍摄物品记忆。
- 定位权限只用于记录拍摄点的经纬度和精度；定位不可用不会阻止保存照片。
- 删除一条记忆时，会同时删除对应原图、缩略图和识别记录。

## 当前限制

- 当前主要面向 Android 手机。
- 安装包是 debug 测试版，不是应用商店正式签名版。
- “位置判断”目前主要是依据画面生成的室内线索，例如“桌面上”或“抽屉里”。GPS 会保存为原始坐标，尚未转换为街道或场所名称。
- 查找会向 MiMo 请求匹配结果，网络较慢时需要稍等片刻。

## 本地运行

首次运行前先安装依赖：

```sh
flutter pub get
```

连接 Android 手机或模拟器后运行：

```sh
flutter run
```

进入 App 右上角设置，填写 MiMo API Key。Key 只保存在本机安全存储里，不会写入源码。

## 打包 APK

生成 debug APK：

```sh
scripts/build_debug_apk.sh
```

APK 输出位置：

```text
build/app/outputs/flutter-apk/小龟寻物-debug.apk
```
